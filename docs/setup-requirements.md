# Setup Requirements

## Prerequisites
- Git for version control
- Hermes Agent (npm install -g @nousresearch/hermes-agent or pip install hermes-agent)
- A game engine (Godot 4, Unity, Unreal 5, or custom)

## First-Time Setup
1. Clone HMGS alongside your game project
2. Start Hermes in your game project directory
3. Tell Hermes: "Read hm-game-studio/FRAMEWORK.md to activate"
4. Tell Hermes: "Read hm-game-studio/skills/start/SKILL.md"

## Directory Structure
The framework expects this layout (created automatically as needed):
design/gdd/ — GDDs
design/ux/ — UX specs
docs/architecture/ — ADRs
production/sprints/ — Sprint plans
production/epics/ — Stories
assets/data/ — Config data
src/gameplay/ — Gameplay code
tests/ — Tests

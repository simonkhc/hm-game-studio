# Directory Structure

## Project Root
```
/                           # Game project root
├── hm-game-studio/         # Framework files (this repo)
├── src/
│   ├── core/              # Engine/framework code
│   ├── gameplay/          # Gameplay systems
│   ├── ai/                # AI behavior
│   ├── networking/        # Multiplayer code
│   ├── ui/                # UI code
│   └── tools/             # Dev tools
├── assets/
│   ├── art/               # Sprites, models, textures
│   ├── audio/             # Music, SFX
│   ├── vfx/               # Particle effects
│   ├── shaders/           # Shader files
│   ├── data/              # JSON config / balance data
│   └── locales/           # Localization files
├── design/
│   ├── gdd/               # Game Design Documents
│   ├── ux/                # UX specifications
│   ├── narrative/         # Story, lore, dialogue
│   ├── levels/            # Level design docs
│   ├── balance/           # Balance data
│   ├── audio/             # Audio design docs
│   └── art/               # Art bible and style guides
├── docs/
│   ├── architecture/      # ADRs and architecture docs
│   ├── performance/       # Performance profiles
│   ├── security/          # Security audits
│   ├── analytics/         # Analytics specs
│   └── postmortems/       # Post-mortems
├── production/
│   ├── sprints/           # Sprint plans
│   ├── epics/             # Epics and stories
│   ├── milestones/        # Milestone tracking
│   ├── releases/          # Release checklists
│   ├── playtests/         # Playtest reports
│   ├── bugs/              # Bug reports
│   ├── retrospectives/    # Retrospectives
│   ├── session-state/     # Session state (gitignored)
│   ├── stage.txt          # Current phase
│   └── review-mode.txt    # Review intensity setting
├── tests/
│   ├── unit/              # Unit tests
│   ├── integration/       # Integration tests
│   ├── performance/       # Performance tests
│   └── helpers/           # Test utilities
└── prototypes/            # Throwaway prototypes
```
